# 津派坞 - 数字藏品展示网站大纲

## 一、网站概述

**网站名称**：津派坞（JINPAIWU）
**定位**：天津文旅数字藏品官方展示与宣传平台
**使命**：焕新天津文旅，打造城市品牌数字名片
**风格**：苹果式简约大气，注重视觉冲击与流畅交互

## 三、首页模块设计

### 1. 导航栏 (Navigation)
- LOGO：津派坞
- 主导航：首页 | 数字藏品 | 资讯中心 | 关于我们
- 右侧：搜索图标 | 用户图标
- 特性：滚动时导航栏背景模糊化，保留核心功能

### 2. 首屏大屏区域 (Hero Section)
- 全屏视频/动态背景：展示天津城市地标建筑（天津眼、劝业场、意式风情街等）
- 主标题：焕新天津文旅，打造城市品牌
- 副标题：探索津派坞，发现独特的数字文化魅力
- CTA按钮：立即探索（滚动至下一区域）
- 特性：视差滚动效果，文字渐入动画

### 3. 核心价值展示区 (Value Proposition)
- 三个卡片并排展示：
  1. **文创赋能** - 将天津传统文化转化为数字艺术品
  2. **珍藏记忆** - 馆藏级数字藏品，永久保存
  3. **资讯动态** - 实时发布最新藏品与活动信息
- 特性：卡片悬停时微微上浮+阴影加深

### 4. 精选藏品轮播区 (Featured Collections)
- 大尺寸卡片展示3-5个精选数字藏品
- 左右滑动导航+自动轮播（8秒切换）
- 每个卡片包含：藏品图片、名称、简介、稀缺度标识
- 特性：滑动时有惯性效果，缩略图预览

### 5. 系列分类入口 (Category Grid)
- 四宫格展示四大系列：
  1. **文创系列** - 天津传统文化创意藏品
  2. **馆藏系列** - 博物馆珍贵藏品数字化
  3. **限定系列** - 节日/活动限定藏品
  4. **新品专区** - 最新上线藏品
- 特性：悬停时图片微缩放+遮罩层滑动出现

### 6. 品牌故事区 (Brand Story)
- 左图右文布局：天津城市风景大图 + 品牌理念文字
- 引用津派坞的愿景与使命
- 特性：滚动触发淡入动画

### 7. 实时资讯区 (Latest News)
- 三列卡片布局展示最新资讯
- 卡片包含：封面图、发布日期、标题、摘要
- 特性：悬停时卡片上浮

### 8. 合作机构展示 (Partners)
- 滚动logo墙展示合作伙伴（博物馆、文创机构等）
- 特性：无限循环滚动效果

### 9. 底部区域 (Footer)
- 四列布局：
  1. 关于我们（品牌介绍、联系方式）
  2. 快速链接（隐私政策、服务条款）
  3. 关注我们（微信公众号、微博、抖音）
  4. 版权信息
- 特性：简洁大气，保持留白

## 七、视觉设计规范

### 1. 色彩方案
- **主色调**：纯白 #FFFFFF
- **辅助色**：浅灰 #F5F5F7（用于背景分区）
- **强调色**：深空灰 #1D1D1F（用于标题和重要文字）
- **点缀色**：金色 #C9A050（用于稀缺度标识、特殊标签）
- **文字色**：
  - 主文字：#1D1D1F
  - 次要文字：#86868B
  - 超链接：#0066CC

### 2. 字体规范
- **主标题**：SF Pro Display / PingFang SC / sans-serif, 600-700 weight
- **副标题**：SF Pro Display / PingFang SC / sans-serif, 500 weight
- **正文**：SF Pro Text / PingFang SC / sans-serif, 400 weight
- **字号**：
  - 超大标题：72-96px
  - 大标题：48px
  - 中标题：32px
  - 小标题：24px
  - 正文：16-18px
  - 辅助文字：14px

### 3. 间距系统
- 基础间距单位：8px
- 页面边距：桌面端 80px，平板 40px，手机 20px
- 区块间距：120px（桌面）/ 80px（移动端）
- 卡片间距：24px

### 4. 动效规范
- **过渡时长**：0.3s - 0.6s
- **缓动函数**：cubic-bezier(0.25, 0.1, 0.25, 1)
- **悬停效果**：上浮8-12px + 阴影加深
- **滚动动画**：元素进入视口时淡入+上移
- **页面切换**：平滑过渡效果

### 1. 技术栈
